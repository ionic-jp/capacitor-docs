# Capacitor: クロスプラットフォームのためのアプリケーションランタイム

<p class="intro">Capacitorはクロスプラットフォームのためのアプリランタイムで、iOS、Android、Electron、<em>ウェブ上</em>でネイティブに動作するウェブアプリを簡単に作ることができる。私たちはこれらのアプリを「Native Progressive Web Apps」と呼び、ハイブリッドアプリを超えた次の進化を表しています。</p>

<img src="/assets/img/docs/capacitor-index.png" style="max-height: 360px" />

## 概要

Capacitorは、一貫性のあるWeb中心のAPIセットを提供し、アプリケーションができる限りWeb標準に近い状態を維持しながら、それらをサポートするプラットフォーム上の豊富なネイティブデバイス機能にアクセスできるようにします。ネイティブ機能の追加は、iOSのためのSwift、AndroidのためのJava、Web用のJavaScript用のシンプルなプラグインAPIを使えば簡単にできます。

Capacitorは[Apache Cordova](https://cordova.apache.org/)や[Adobe PhoneGap](https://phonegap.com/)のスピリチュアルな後継であり、[React Native](http://facebook.github.io/react-native/) や [Turbolinks](https://github.com/turbolinks/turbolinks)のような他の人気のクロスプラットフォームツールから着想を得ているが、最新のウェブアプリをすべての主要プラットフォーム上で容易に実行できるようにすることに焦点を絞っています。Capacitorには、多くの既存のCordovaプラグインに対する下位互換性のあるサポートがあります。

## はじめよう

Capacitorの使用は簡単です。Capacitorは、既存の最新のJavaScriptプロジェクトへのドロップ・インとして使用することも、新規のCapacitorプロジェクトをゼロから作成するために使用することもできます。
