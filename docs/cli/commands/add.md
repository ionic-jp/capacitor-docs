# Capacitor CLI - cap add

あなたのプロジェクトに Native プラットフォームを追加します。

```bash
npx cap add <platform>
```

<strong>Inputs:</strong>

- `platform` (required): `android`, `ios`

<strong>Options:</strong>

iOS Only:
- `--packagemanager`: `SPM`, `Cocoapods`
